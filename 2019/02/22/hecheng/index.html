Java中类的合成与继承

By zzulixjh

Published 2019-02-22

Contents
  1. 1. 合成
  2. 2. 继承
  3. 3. 总结

我们通过创建新类来重复使用代码，但却用不着重新创建，可以直接使用别人已建好的现成类。
第一个最简单的方法:在新类里简单的创建原有类的对象，我们把这种方法叫做”合成”，因为新类由现有类的对象合并而成。我们只是简单地重复利用代码的功能，而不是采用它的形式。
第二种是创建一个新类，将其作为现有类的一个”类型”。我们可以采取现有类的形式，并在其中加入新代码，同时不会对现有类产生影响。这种行为叫做”继承”。

合成


为进行合成，我们只需在新类里简单地置入对象句柄即可。
对象句柄会初始化为null。
所谓合成，就是在一个新的类里对原有的几个现成的类进行调用。

继承


继承的语法:
public class A entends B{

}
这样在A里就可以调用B类的东西了。
可以允许多重继承，即B类继承了A类，C类再继承B类的时候，C也继承了A。
所谓继承，就是选择一个现有的类，然后制作一个它的特殊版本。
继承是对一种特殊关系的表达，意味着”这个新类属于那个旧类的一种类型”。

总结


无论继承还是合成，我们都可以在现有类型的基础上创建一个新类型。但在经典情况下，我们通过合成来实现现有类型的”再生”或”重复使用”，将其作为新类型基础实施过程的一部分使用。但如果想实现接口的”再生”，就应该使用继承。
