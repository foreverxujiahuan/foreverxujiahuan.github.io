    • Vector

    创建


    Vector可实现自动增长的对象数组。
    创建一个整型数组的方法:
    Vector arr = new Vector<>();

    插入


    插入一个整型对象
    arr.addElement();
    指定位置插入对象
    arr.add(index,element)
    arr.insertElementAt(obj,index)

    删除


    删除指定位置元素
    arr.remove(int index)
    删除数组中的指定对象(该方法会删除第一个元素)
    arr.removeElement(object)
    删除指定位置的元素
    arr.reomve(index)
    删除所有元素
    arr.removeAllElements()

    查询


    查询元素所在位置
    arr.indexOf(index)
    查询指定位置的元素
    arr.get(index)

    获取Vector的一些属性


    获取长度
    arr.size()
    获取数组所有元素
    arr.toString()

    Stack


    创建一个Stack
    Stack stack = new Stack<>();
    压入元素:
    stack.push(element)
    弹出元素:
    stack.pop()
    注意Stack属于一种Vector，因此对于Vector可用的方法均可对Stack使用。

    Hashtable


    创建哈希表
    Hashtable hs = new Hashtable()
    添加元素
    hs.put(key,value)
    获取元素
    hs.get(key)
