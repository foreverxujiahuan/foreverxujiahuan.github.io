关于int[]类型数组

一维数组的声明方式


以声明一个长度为5，数据类型为Int的数组arr为例
方式一:int[] arr = new int[5]
方式二:int arr[] = new int[5]

数组的初始化


方式一:定义数组arr时就给他赋值
int[] arr = {1,2,3,4,5}
方式二:给指定位置的元素依次赋值
arr[0]=1;
arr[1]=2;
arr[2]=3;
arr[3]=4;
arr[4]=5;

数组的引用


获取位置为index的数据
arr[index]

Arrays

填充数组fill


1.Arrays.fill(arr,n)
给arr中所有的值全变为n
2.Arrays.fill(arr,start,end，n)
将arr中区间为[start,end)的数据填充为n

数据排序sort


给数组arr进行排序(升序)
Arrays.sort(arr)
给指定区间[start,end)的数据排序
Arrays.sort(arr,start,end)

比较数组是否相等


Arrays.equals(arr1,arr2)

寻找指定元素值的下标index


Arrays.binarySearch(arr,value)
总结:Arrays主要用于操作已经存在的数组

List


List接口是Collection接口的子接口，List有一个重要的实现类–ArrayList类，List中的元素是有序排列的而且可重复，所以被称为是序列。

List可以精确的控制每个元素的插入位置，或删除某个位置元素，它的实现类ArrayList底层是由数组实现的。

List中有增删改查的方法
这里以ArrayList为例

创建一个list


List list = new ArrayList<>();

增


向数组末尾添加元素n
list.add(n)
向指定位置添加元素
list.add(index,n)

删


移除位置为index的元素
list.remove(index)

改


将位置为index的元素改为n
list.set(index,n)

查


获取位置为index的元素
list.get(index)
获取元素为value首次出现的位置
list.indexOf(value)
获取元素为value最后一次出现的位置
list.lastindexOf(value)
获取数组长度
list.size()

LinkedList


LinkedList是链表类，采用链表结构保存元素。链表结构的优点是便于向集合插入和删除元素。因为在插入或删除元素时，不需要移动任何元素。但若需要随机访问集合中的元素时，LinkedList集合就显得比较慢了。
