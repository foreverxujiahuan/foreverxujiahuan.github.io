11. Container With Most Water


Given n non-negative integers a1, a2, …, an , where each represents a point at coordinate (i, ai). n vertical lines are drawn such that the two endpoints of line i is at (i, ai) and (i, 0). Find two lines, which together with x-axis forms a container, such that the container contains the most water.

Note: You may not slant the container and n is at least 2.

Example:

Input: [1,8,6,2,5,4,8,3,7]
Output: 49
问题分析:定义两个指针i、j，一个指向数组第一个，第二个指向数组最后一个，两个同时开始遍历，对于左边的指针i，如果右边的指针对于的值height[j]>height[i]，那么第i个位置的最大可能性就是height[i]*(j-i)了。对于右边的值也是一样。这样O(n)时间复杂度内就可以遍历完了。

12.Integer to Roman


Roman numerals are represented by seven different symbols: I, V, X, L, C, D and M.

Symbol Value
I 1
V 5
X 10
L 50
C 100
D 500
M 1000
For example, two is written as II in Roman numeral, just two one’s added together. Twelve is written as, XII, which is simply X + II. The number twenty seven is written as XXVII, which is XX + V + II.

Roman numerals are usually written largest to smallest from left to right. However, the numeral for four is not IIII. Instead, the number four is written as IV. Because the one is before the five we subtract it making four. The same principle applies to the number nine, which is written as IX. There are six instances where subtraction is used:

I can be placed before V (5) and X (10) to make 4 and 9.
X can be placed before L (50) and C (100) to make 40 and 90.
C can be placed before D (500) and M (1000) to make 400 and 900.
Given an integer, convert it to a roman numeral. Input is guaranteed to be within the range from 1 to 3999.

Example 1:

Input: 3
Output: “III”
Example 2:

Input: 4
Output: “IV”
Example 3:

Input: 9
Output: “IX”
Example 4:

Input: 58
Output: “LVIII”
Explanation: L = 50, V = 5, III = 3.
Example 5:

Input: 1994
Output: “MCMXCIV”
Explanation: M = 1000, CM = 900, XC = 90 and IV = 4.
题目分析:
创建一个数字数组和一个字符串数组，把每个可能的字符串对应的值的可能性都列出来，然后两个数组下标对应，然后开始遍历数值的那个数组，只要当前的数字比该位置的值大，就向结果中添加该位置对应的字符串。

X can be placed before L (50) and C (100) to make 40 and 90.

14.Longest Common Prefix


Write a function to find the longest common prefix string amongst an array of strings.

If there is no common prefix, return an empty string “”.

Example 1:

Input: [“flower”,”flow”,”flight”]
Output: “fl”
Example 2:

Input: [“dog”,”racecar”,”car”]
Output: “”
Explanation: There is no common prefix among the input strings.
Note:

All given inputs are in lowercase letters a-z.

15.3Sum


Given an array nums of n integers, are there elements a, b, c in nums such that a + b + c = 0? Find all unique triplets in the array which gives the sum of zero.

Note:

The solution set must not contain duplicate triplets.

Example:

Given array nums = [-1, 0, 1, 2, -1, -4],

A solution set is:
[
[-1, 0, 1],
[-1, -1, 2]
]
问题分析:
这道题也得定义两个指针，分别从数组的头上和尾巴上同时开始寻找第三个数。先对数组进行排序，然后遍历数组，对于数组中的每一个数字，我们都得要寻找两个数相加等于这个数的负数，并且这两个数不能是该数字。针对每一个数字，我们都定义两个指针left和right,如果两个位置的和加起来小于要找的数字，那么left+1，如果大于要找的数字，那么right-1。
