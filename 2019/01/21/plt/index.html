因为这几天准备比赛的原因，我把Matplotlib做了一个简单的总结。

导入


我们平时所用的matplotlib基本就是matplotlib中的一个模块，即pyplot,我们基本会使用这一行代码完成导入:import matplotlib.pyplot as plt

直线图的绘制


直线其实就是两个点的连线，因此只要两个点确定了，就可以确定一条直线。
demo:
画一条y=x+1的直线
确定点(0,1)和(1,2)，然后使用plt.plot()命令。
import matplotlib.pyplot as plt
x = [0,1]
y = [1,2]
plt.plot(x,y)
plt.show()

也可以想到折现图的画法，折线图无非就是多了几个点，然后将这几个点一个一个连起来。
demo:
import matplotlib.pyplot as plt
x = [0,1,2]
y = [1,2,4]
plt.plot(x,y)
plt.show()

曲线图的画法


在二维平面中，要确定一条曲线，就是要确定函数和变量的范围。我们需要借助numpy函数来完成。
先来介绍一下arange的用法
arange(start,stop,step,dtype),这个函数里一共需要填入四个参数，第一个是起始值，第二个是终止值，第三个是中间间隔的大小，最后一个是数据类型。
举个例子，我们来画一条二次函数的曲线图。
import matplotlib.pyplot as plt
import numpy as np
x = np.arange(0,10,0.01)
y = x2
plt.plot(x,y)
plt.show()

若要画其他曲线，只要把y改成其他函数即可。
当然也可以用linespace来进行自变量的选取。
import matplotlib.pyplot as plt
import numpy as np
x = np.linspace(1,10)
y = x2
plt.plot(x,y)
plt.show()

对画的图添加注释


import matplotlib.pyplot as plt
import numpy as np
x = np.linspace(1,10)
plt.xlabel(“x”)
plt.ylabel(“y”)
plt.title(“y=x^2”)
y = x**2
plt.plot(x,y)
plt.show()
