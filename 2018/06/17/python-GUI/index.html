什么是GUI编程


GUI是Graphical User Interface的缩写，即对图形用户界面的编写。

Tkinter


Tkinter是Python的默认GUI库。因为Tkinter在系统中不是默认必须安装的，我们需要先安装Tkinter，在Pycharm中只需在setting中导入该模块就可以，很方便。
导入后我们可以检测是否导入成功，win+r,输入cmd,调到python模式，import Tkinter，没有报错就说明导入成功了。
