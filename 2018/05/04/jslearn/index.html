一、基础语法
1.JS中用关键字var来定义变量，使用等号来为变量赋值:
例如:var x,length
x = 5
length = 6
2.js中的注释
我们使用双斜杠来注释，例如
//注释的内容
3.元素的获取与输出
下面的javascript语句向id=”demo”HTML元素输出文本”你好Dolly”
document.getElementById(“demo”).innerHTML = “你好 Dolly”;
4.弹窗
JS使用alert(“输出内容”)来进行弹窗
二、JS对象
定义Javascript对象可以跨越多行，空格跟换行不是必须的:
比如我们定义一个人的对象
var person = {
firstName:”John”,
lastName:”Doe”,
age:50,
eyeColor:”blue”
};
我们可以通过两种方式访问对象属性
person.lastName;
person[“lastName”];
三、JS事件
我们可以定义一个方法(事件)来给其他文本进行调用
demo:
我们定义一个函数
function changetext(id)
{
id.innerHTML=”Ooops!”;
}
如果一个h1标签的文本里有 onclick=”changetext(this)”的话，点击后就会出现Ooops!字样。
