今天来聊聊几款我用过编辑器。

一款好的编辑器是开发人员的必备的，当然编辑器没有决对的还坏之分。不同的工程师会喜欢不同的编辑器。比如C、C++都会喜欢用codeblocks,Python喜欢用Pycharm,前端喜欢用HBuilder等等。下面我来分别介绍一下。

Sublime Text3:


这是我最早接触前端时学长让我用的一款编辑器，我记得我一直用到了七月份吧，才改用vscode的。这是一款轻型的编辑器，因为本身的大小很小，所以打开的速度很快。而且黑色的背景一般男生都会比较喜欢。

HBuilder


这款编辑器听说是前端的第一选择，我的前端实验室的学姐就让我们用Hbuilder.我们前端实验室貌似除了我都在用这款编辑器，但是我对于这款编辑器真的是知之甚少，也没有用它来写过一个项目，就记得去年三月份好像用它写过一个课程表之后就很少接触了。

Notepad++


这也是一款我很少用的编辑器，甚至我连我什么时候下的都忘记了。但是因为我的php文件默认是用Notepad++打开的，所以有时候临时改一些文件也会用到它。对于它同样我也不是很熟悉，所以就不做评论了

codeblocks


相信这款软件大多数理工科的学生都挺熟悉的，因为他是一款很好的C语言编辑器。记得我大一的时候C语言很差，上课也是基本不听的，所以对这款编辑器有点反感…但是大二我也在抽空补C语言，为我以后学习算法打基础，总之感觉用它来写C还是不错的，我还特地给他换了个背景颜色。

vs code


vs code是我现在主要用的一款编辑器。只能说这真的是一款非常强大的编辑器。这款软件给我的感觉就是酷酷的，不同的语言脚本它会有不同的小图标来显示，比如HTML文件它是用一个橙色的括号，PHP它是用一只大象来表示，都非常形象。而且它的背景颜色，不同代码的颜色，我都感觉非常好，给人一种美感。

Pycharm


由于前一段时间学了会儿Python，就下载了这个Python中最著名的编辑器–Pycharm。但是这个编辑器个人感觉太大了，800多MB的软件像我这种破电脑打开起来的速度是相当慢的。而且感觉这款软件配置起来也挺麻烦的，记得我刚下的时候配置就弄了我一个多小时。毕竟是大软件嘛，想往Python方向发展的人可以多多研究研究
