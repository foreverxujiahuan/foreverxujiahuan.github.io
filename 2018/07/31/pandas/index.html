准备工作

pandas安装


直接使用命令 pip install pandas
如果有问题就去下载wheel文件进行安装

pandas导入


因为pandas一般会和numpy配合使用，所以一般会先导入numpy
import numpy as np
import pandas as pd
和numpy一样，一般会把pandas简写作pd

pandas能用来做什么


pandas是一款很好的数据分析工具，掌握好pandas、numpy、matplotlib可以为数据挖掘打下良好的基础。

pandas的两个主要数据结构


要使用pandas,你首先就得熟悉它的两个主要数据结构:Series和DataFrame。虽然它们并不能解决所有的问题，但它们为大多数应用提供了一种可靠的、易于使用的基础。

Series


Series由一组数据以及一组与之相关的数据标签（即索引）组成
由一组数据即可产生最简单的Series,在没有为数据指定索引的情况下，会自动创建一个0到N-1的整数型索引。举个例子，我创建一个Series([12,14,16,17])，那么它的样子就是这样的

可以通过values和index属性获取其数组表示形式和索引对象
当然我们也可以指定索引,例如Series([4,7,-5,3],index=[‘d’,’b’,’a’,’c’])

可以将字典直接改成Series,例如sdata是一个字典，那么只需Series(sdata)，他就被转换成了Series
Series有用name属性，可以手动给他加，例如针对一个obj，可以使用obj.name = “”来改变它的name，可以通过obj.index.name=””来改变它的索引名字
与Numpy数组相比，我们可以通过索引的方式选取Series中的单个或一组值
可以将Series看成是一个定长的有序字典，因为它是索引值到数据值的一个映射。
Series索引的工作方式类似于Numpy数组,只不过Series的索引值不只是整数。例如对于名为obj的Series,可以使用obj[1]来获取index排序为1的值，可以使用obk[‘a’]来获取index名字为a的值，在索引的时候，和numpy差不多，比如obj[2:4]就是获取index排序为2和三的值，当然也可以传入多个index名称，例如obj[‘a’,’b’]就是获取行名为a和b的值
两个Series相加的时候，自动的数据对其操作在不重叠处引入了NA值
可以使用unique函数获取Series中的唯一值,如obj.unique()

DataFrame


DataFrame是一个表格型的数据结构。
DataFrame既有行索引也有列索引，它可以被看作由Series组成的字典
最常用的构建DataFrame的方法就是直接传入一个由等长列表或NumPy数组组成的字典,这里将data强行转换成DataFrame:


和Series一样，DataFrame会自动加上索引，且全部列会被有序排列
可以将DataFrame的列表获取为一个Series，例如获取state这一行
就可以将在原来的DataFrame后面加上[‘state’]
我们还可以在后面直接加上一个列，例如我要在一个名为frame的DataFrame后面加上一个debt列，其值全为16.5，那么我只需frame[‘debt’]=16.5
可以将DataFrame的列获取为一个Series,比如frame2.year或者frame[‘year’]，就是获取了frame2中的year列
行也可以通过位置或名称的方式进行获取，比如使用索引ix，frame2.ix[],括号里面可以填行的数字或者是行的名称.
可以使用del对一些列进行删除，比如我要删除名为test的列，只需 del frame[‘test’]
也可以用columns，获取DataFrame的列，如frame.columns
也可以对frame进行转置，例如frame.T
drop方法返回的是一个在指定轴上删除了指定值的新对象
例如我要删除行名为a的行，只需frame.drop(‘a’)
DataFrame的对其操作会同时发生在行和列上，例如df1和df2相加，只有他们的行和列全都相同的元素才会相加，匹配不到的地方会产生NAN值
也可以使用add方法,比如我要让df1和df2相加，匹配不到的地方的值用0替代，只需写作df1.add(df2,fill_value=0)

排序和排名


要对行或列索引进行排序，可使用sort_index方法，它将返回一个已排序的新对象
对于DataFrame,可以根据任意一个轴上的索引进行排序sort_index()和sort_index(axis=1)

处理缺失数据


pandas使用浮点值NaN表示浮点和非浮点数组中的缺失数据
Python内置的None值也会被当作NA处理
可以使用dropna()方法过滤掉缺失数据
在填充缺失数据的时候，对于大多数情况而言，fillna方法是最主要的函数。通过一个常熟调用fillna就会将缺失值替换为那个常数值
比如df.fillna(0)就可以将原来的DataFrame中的NAN值变为0
