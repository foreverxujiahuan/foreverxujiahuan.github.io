安装BeautifulSoup


推荐直接用命令安装
pip install BeautifulSoup4
如果命令报错，无法解决，可以尝试使用wheel安装，从网上下载wheel文件，然后使用pip安装
https://pypi.python.org/pypi/beautifulsoup4

节点选择器

选择元素


在上面的基础上，我们可以提起我们想要的一些元素
比如我要提取网页中的title部分就是soup.title
网页中的head部分就是soup.head

提取信息


我们可以用name属性获取节点的名称
比如我们要获取title的节点的名称，以上面的文本为例，就是
soup.title.name
运行结果就是title

获取属性


每个节点可能有多个属性，比如id和class等，选择这个节点元素后，可调用attrs获取所有属性:
比如我要获取p的属性，只需
soup.p.attrs即可
如果要获取p节点的class，只需
soup.p.attrs[‘class’]
可以使用string属性获取节点元素包含的文本内容
比如要获取第一个p系欸但的文本:
soup.p.string

关联选择


在做选择的时候，有时候不能做到一步就选到想要的节点元素，需要选中某一个节点元素，然后以它为基准再选择它的子节点、父节点、兄弟节点等。
1.子节点和子孙节点
选取节点元素之后，如果想要获取它的直接子节点，可以调用contents属性,比如我要调用head的子节点，那么我只需soup.head.contents
需要注意的是，contents属性得到的结果是直接子节点的列表，并不会把孙子节点单独选出来。
如果想要得到所有子孙节点的话，可以调用descendants属性
比如soup.p.descendants
2.父节点和祖先节点
如果要获取某个节点元素的父节点，可以调用parent属性:
比如要获取a节点的父节点，只需soup.a.parent
需要注意的是，这里仅仅获取的是a节点的父节点元素，没有再往外寻找。
如果想获取所有的祖先节点，可以调用parents属性
soup.a.parents即可获取所有的祖先节点
3.兄弟节点
可以用next_sibling和previous_sibling分别获取节点的下一个和上一个兄弟元素，next_siblings和previous_siblings则分别返回所有前面和后面的兄弟节点的生成器。

方法选择器

find_all()


find_all,顾名思义，就是查询所有符合条件的元素。给它传入一些属性或文本，就可以得到符合条件的元素，它的功能十分强大。
它的API如下:
find_all(name,attrs,recursive,text,**kwargs)

name


比如我们现在需要获取所有的ul标签的内容
就是soup.find_all(name=’ul’)
这里我们调用了find_all方法，传入name参数，其参数值为ul。也就是说，我们要查询所有ul节点。

attrs


除了根据节点名查询，我们也可以传入一些属性来查询
比如我们要获取id为list-1的节点，我们可以
soup.find_all(attrs={‘id’:’list-1{})

text


text参数可用来匹配结点的文本，传入的形式可以是字符串，可以是正则表达式对象，示例如下:
比如我现在要找到拥有文本为Hello的文本，那么我只需
soup.find_all(text=re.compile(‘Hello’))
就是拥有Hello文本的那一段完整的文本了

find()


除了find_all方法，还有find()方法，只不过该方法返回的是单个元素，也就是第一个匹配到的元素，而前者返回的是所有匹配的元素组成的列表

CSS选择器


使用CSS选择器时，只需要调用select()方法，传入相应的CSS选择器即可，示例如下:
比如我们要选区class为panel的元素，我们只需
soup.select(‘.panel’)
我们要选区所有ul下的li节点，我们只需
soup.select(‘ul li’)
