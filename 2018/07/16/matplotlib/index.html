使用matplotlib和numpy实现简单的数据可视化

By zzulixjh

Published 2018-07-16

Contents

在开始使用他们绘图之前，我们需要先对matplotlib和numpy进行简单的学习。
我以一个coursera上的机器学习的一个房价预测问题为例，看一下如何利用python进行简单的数据可视化。

需要注意的是这里的prices.txt文件需要自行去下载。
下载地址:https://github.com/carefree0910/MachineLearning/edit/master/_Data/prices.txt
接下来一句一句的看一下这代码:
import numpy as np和 import matplotlib.pyplot as plt就是导入这两个模块，并且给它们重新命名为np和plt
x,y = [],[]就是定义了两个存储输入数据(x)和目标数据(y)的数组
接下来就是遍历数据集，这里我们用open函数以只读的方式打开了prices.txt文件，然后将里面的数据以”,”为分隔符分开，分别赋值给_x和_y,接下来我们就得把刚刚获取到的_x和_y添加到数组x和y中
接着我们将python中自带的list转化为numpy模块中的array，即x,y=np.array(x),np.array(y)
这里因为数值比较大，所以我们这里将对x做简单的处理来降低问题的复杂度。在这个例子里，我们将采取常用的将输入数据标准化的做法，即x = (x-x.mean())/x.std()
其中x.mean()就是x的平均值的意思，std(x)代表x的标准差。另外这行代码的意思是将数组x中的每个数字进行该条语句的复制操作。这条语句执行完毕之后将会得到一个新的数组x
plt.figure()是创建画布的意思
另外这里用到了scatter函数，这里的参数x,y分别是横纵坐标，c是颜色，这里的g就是指绿色的意思，s是点的大小的意思，s越大，点越大
最后plt.show就是画出图的意思了。
