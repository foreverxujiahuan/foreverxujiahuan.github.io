在我们用numpy和matplotlib完成简单的数据可视化后，就可以真正进入机器学习了。
机器学习算法很多，但是通常而言，进行机器学习的过程会包含三步:
1.获取与处理数据
2.选择与训练模型
3.评估与可视化结果

获取与处理数据


这一步我就不多说了，有不清楚的可以去看我的”使用matplotlib和numpy实现简单的数据可视化”文章。


到这一步为止，我们已经将需要获取的.txt文件中的数据呈现在了一张图片中。

选择与训练模型


在弄好数据之后，下一步就要开始选择相应的学习方法和模型了。通过可视化数据，可以非常直观的感受到:很有可能通过线性回归中的多项式拟合得到一个不错的结果。
对于大多数机器学习算法，所谓的训练正是最小化某个损失函数的过程。
这里需要解释两个Numpy中带的函数:
polyfit(x,y,deg):该函数会返回是的损失函数最小的参数p，换句话说，该函数就是模型的训练函数。
ployval(p,x):根据多项式的各项系数p和多项式中x的值，返回多项式的值y。

评估与可视化结果


我们定义一个函数，根据参数n，输入的x、y返回相对应的损失
再定义测试参数集并根据它进行各种实验
最后在输出响应的损失

最后再画出图像


其中三条线分别代表n=1,n=4,n=10的情况
下面给出完整代码
import numpy as np
import matplotlib.pyplot as plt
from matplotlib.pyplot import plot,savefig
x,y = [],[]
for sample in open(“prices.txt”,”r”):
_x,_y = sample.split(“,”)
x.append(float(_x))
y.append(float(_y))
x , y = np.array(x),np.array(y)
x = (x-x.mean())/x.std()
plt.figure()
plt.scatter(x,y,c=”g”,s=6)

#plt.show()
x0 = np.linspace(-2,4,100)
def get_model(deg):
return lambda input_x=x0: np.polyval(np.polyfit(x,y,deg),input_x)
def get_cost(deg,input_x,input_y):
return 0.5((get_model(deg)(input_x)-input_y)*2).sum()
test_set = (1,4,10)
for d in test_set:
print(get_cost(d,x,y))
plt.scatter(x,y,c=”g”,s=20)
for d in test_set:
plt.plot(x0,get_model(d)(),label=”degree = {}”.format(d))
plt.xlim(-2,4)
plt.ylim(1e5,8e5)
plt.legend()
plt.show()
至此，我的第一个机器学习样例就完成了。
